# 프리코스 2주차 자동차경주게임

## 기능 요구사항

- [ ] 자동차
  - [ ] 이름을 가지고 있음
  - [ ] 전달 받은 숫자가 4이상일 경우 전진, 3이하일 경우 멈춤
- [ ] 자동차 리스트
  - [ ] 일급 콜렉션
  - [ ] 자동차 이름을 전달 받아 자동차 리스트에 등록
  - [ ] 각 자동차에게 0 ~ 9 랜덤 숫자 전달
- [ ] 우승자
  - [ ] 일급 콜렉션
  - [ ] 우승자 등록
  - [ ] 우승자 리스트 가져오기
- [ ] 게임
  - [ ] 사용자에게 자동차 이름 리스트를 전달 받아 자동차 리스트에 등록
  - [ ] 사용자에게 시도 횟수를 입력 받아 실행 결과 출력
- [ ] 입/출력
  - [ ] 자동차 이름 입력
  - [ ] 시도 횟수 입력
  - [ ] 우승자 출력
- [ ] 공통
  - [ ] 예외 처리 명확하게
